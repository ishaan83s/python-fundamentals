# Python Fundamentals 🐍

A day-by-day journal of learning Python from the ground up to covering core syntax, data structures, control flow, functions, recursion, file handling, and object-oriented programming, along with mini projects and bonus "side quest" topics.

## 📖 About

This repository documents my Python learning journey, organized into daily folders (`day01_learning` → `day16_learning`). Each day builds on the previous one, moving from basic syntax to more advanced concepts like OOP and exception handling. Alongside the daily lessons, there are standalone mini projects and side quests exploring useful Python features in more depth.

## 📚 Topics Covered

- **Basics:** variables, input/output, operators, print formatting
- **Strings:** string operations and built-in string functions
- **Conditionals:** if/elif/else, nested conditions
- **Data Structures:** lists, tuples, dictionaries, sets (with slicing and built-in methods)
- **Loops:** `for` loops, `while` loops, `break`/`continue`, range-based iteration
- **Functions:** basics, default parameters, recursion
- **File Handling:** read, write, append modes (`r`, `w`, `a`, `r+`, `w+`, `a+`)
- **Object-Oriented Programming:**
  - Classes, objects, and constructors
  - Encapsulation, abstraction, inheritance, and polymorphism (the 4 pillars)
  - Static methods, class methods, private members, `del` keyword
  - Operator overloading and dunder methods
  - `property` and `super()`
- **Exception Handling:** try/except, `finally`, raising and defining custom exceptions
- **Extras:** docstrings, f-strings

## 🛠️ How to Run

Each concept lives in its own `.py` file. To run any script:

```bash
python3 path/to/file.py
```

For example:
```bash
python3 day07_learning/functions_basics.py
```
